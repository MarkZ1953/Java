DecimalFormat es una clase en Java que se utiliza para formatear números en un formato de salida específico. Permite controlar la precisión y 
el redondeo, así como la forma en que se separan los miles, los decimales y el símbolo del signo.

DecimalFormat pertenece al paquete java.text y es una subclase de NumberFormat. Para utilizar DecimalFormat, se debe crear una instancia de la clase y 
especificar el patrón de formato deseado. El patrón se establece mediante una cadena que contiene caracteres especiales que representan los distintos 
elementos del formato.

- Patrones : 
- "#,###.00" | Indica que el numero debe ser formateado con separadores de miles y dos decimales.
